# EventOS Frontend

This directory contains the frontend application for the EventOS project. 
The actual React code is located inside the `eventos` folder.

## How to run the frontend

Since the project files are located inside the `eventos` subdirectory, you need to navigate there first.

1. **Navigate into the project directory**:
   ```bash
   cd eventos
   ```

2. **Install dependencies** (you only need to do this once):
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the app**:
   Open your browser and navigate to the URL shown in your terminal (usually `http://localhost:5173`).
